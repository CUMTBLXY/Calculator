js数据类型

1.字符串
var str='string';
var str2 = new String('sss');

2.数字
var i=100;
var i1 = new Number(32);

3.布尔
var flag=true;
var flag2 = new Boolean(true);

4.数组
var array1 = [];
var array2 = new Array();

5.对象
var obj = {};
var obj1 = new Object();

6.Null Undefined
var a = null;
var b = undefined;

定义变量如果是原始数据类型 推荐使用字面量 不推荐使用new关键字
